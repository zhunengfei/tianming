# 内置界面字体

这些字体用于“设置 -> 主题字号”里的内置字体选项，随游戏本体、热更新包和网页端资源一起分发。

- `ZCOOLXiaoWei-Regular.ttf`：ZCOOL XiaoWei，SIL Open Font License 1.1。
- `ZCOOLQingKeHuangYou-Regular.ttf`：ZCOOL QingKe HuangYou，SIL Open Font License 1.1。
- `MaShanZheng-Regular.ttf`：Ma Shan Zheng，SIL Open Font License 1.1。
- `LongCang-Regular.ttf`：Long Cang，SIL Open Font License 1.1。

来源为 Google Fonts / jsDelivr 镜像。以后新增字体时，应同时在 `tm-theme-font.js` 的 `FONT_OPTIONS` 与 `styles.css` 的 `@font-face` 中登记。
