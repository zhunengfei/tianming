## §1·6 共性轴·1300 年不变量

科举制·从隋开皇九年 (587) 初设进士科到清光绪三十一年 (1905) 废止·跨 1318 年·改朝换代 9 次·制度本体却保留 6 个**始终成立**的轴。这 6 轴是 v7 设计**所有 slice 必须套上的 paradigm checker**·若 slice 设计违反任一轴的历史共性·必反 audit。

### §1.A·权力博弈·皇权 vs 文官集团 vs 地方·三向指针

科举从诞生第一天起·就是**三向博弈的工具**·而非单纯"选才"。三个利益方·

1. **皇权**·想用科举绕过门阀世家·把任官权收归中央 (隋唐初衷)·想用殿试钦点 + 改卷 + 内定固化"天子门生"
2. **文官集团**·想用科举固化"士大夫共治"·主考-门生网络是其再生产机制 (唐宋座师制 / 明清房师制)
3. **地方士族 / 豪强**·想用解额 + 行卷 + 冒籍 + 捐纳侵蚀中央配额·维持地方科第世家 (唐五姓七家·宋苏氏三世·清桐城方氏六代)

这 3 方·**任何朝代任何时点都同时存在**·只是强弱不同。Slice 设计若只考虑"皇权 vs 文官"两向·必踩坑·因为地方侵蚀是**长期最大变量** (东汉袁绍门生故吏遍天下·唐末藩镇·明末江南文社·清末团练)。

**4 大侵蚀路径**·每个朝代都有·只是 toggle 不同·

| 路径 | 机制 | 隋 | 唐 | 北宋 | 南宋 | 元 | 明 | 清 |
|---|---|---|---|---|---|---|---|---|
| 恩荫 | 高官子直补官 | off | **on** (五品以上) | **on** (七品以上) | on | on | on (五品以上 + 限) | on (限) |
| 捐纳 | 银钱换贡生 / 监生 / 知县 | off | (战时) | (战时) | (战时) | (元代官买) | (土木堡后) | **on** (常制·清末四成) |
| 行卷 | 考前递文章给主考 | off | **on** (唐风) | off (糊名后) | off | off | off | off |
| 冒籍 | 占外地解额 | (? 待考) | on (江南占长安) | on (北方占南方) | on | on (汉人占色目) | **on** (江南占北方) | **on** (奉天 / 顺天激烈) |

**v7 实现锚**·权力博弈不是 GM 顶级 var·而是**三向 tension 派生指标**·
- 皇权·`P.imperial.huangwei` (现存)·受殿试钦点 / D4 宦官 / 反舞弊 toggle 影响
- 文官·`GM._factionTension` (Slice C2·v6.5 已建)·受主考派系 / 门生网络密度影响
- 地方·新建·**`P.keju._localPressure`** (Slice K3 待定)·受解额公平度 / 冒籍率 / 捐纳占比影响
- 三向闭环·Slice J1 F1/F2/F3 三指针的**底层就是这三向**·F1 = 皇权吸纳·F2 = 文官循环·F3 = 地方整合

### §1.B·筛选结构·N 级状态机·人数指数衰减·主考从地方到皇帝

任何朝代·科举都是**漏斗状 N 级状态机**·从底层"应考人"到顶层"进士"·**每级人数衰减一个数量级**·主考从地方 (县 / 州) 逐级上升到中央 (礼部 / 皇帝)。

通式·`tier_n+1.count = tier_n.count × (1/10 到 1/30)`·
- 隋 3 级·州试 (万) → 礼部试 (千) → (无殿试)·进士数十
- 唐 3 级·州试 (千-万) → 礼部试 (千) → 吏部释褐试 (百)·进士 20-30
- 北宋 3 级 (1065 后)·解试 (10 万+) → 省试 (千-万) → 殿试 (200-400)
- 明清 6 级·童试 → 院试 (秀才) → 乡试 (举人) → 会试 (贡士) → 殿试 (进士) → 朝考 (庶吉士)·秀才 50 万 → 进士 200-300

**关键不变量**·
1. **底层数量级**·应考人数始终 5-50 万·稳态
2. **录取率**·会试录取率 < 5%·殿试 ≈ 100% (走过场)·一甲只 3 人
3. **金字塔顶**·状元 = 头甲第一·**每 3 年 1 人**·1300 年累计 ~700 人 (隋唐宋元 + 明清 552 名·有名录)
4. **主考升级**·下层主考是地方官 (县令 / 知府 / 学政)·上层主考是中央官 (礼部尚书 / 翰林 / 总督)·顶层主考是皇帝本人 (殿试)

**v7 实现锚**·Slice B1/B2/D1-D5 严格按此 6 级走·不可压缩 (压缩成 3 级 = 失真)·但 v7 选择**抽象到 3-tier 简化** (院试 = "州试" / 乡试 + 会试 = "省试" / 殿试 = "殿试")·跨朝代用·tier 数差异在 scenario 配·见 §2。

### §1.C·主考官·派系 / 学派 / 籍贯 / 风格·座师-门生网络

主考·**不只是阅卷人**·是**派系网络的节点**。任何朝代都有 4 维属性·

| 维度 | 含义 | 历史名例 |
|---|---|---|
| **派系** (party) | 朝堂党争归属 | 唐李德裕 (李党)·宋王安石 (新党)·明顾宪成 (东林) |
| **学派** (school) | 学术流派 | 北宋程颐 (理学)·南宋朱熹 (闽学)·明王阳明 (心学)·清戴震 (考据) |
| **籍贯** (region) | 地缘集团 | 唐"关陇" / "山东"·宋"川党" / "洛党" / "朔党"·明"江南" / "北党" |
| **风格** (style) | 取士偏好 | 唐韩愈"古文" / 李商隐"骈俪"·宋苏轼"才气" / 王安石"经术"·明 八股工整 |

**座师-门生**·主考称"座师"·考生中举称"门生"。门生关系·**终生不解·跨朝代延续**·

- 东汉·门生故吏制 (袁绍门生故吏遍天下)
- 唐·座主门生·"白头门生"称谓·主考死后门生披麻 (唐李宗闵-牛僧孺)
- 北宋·欧阳修主考嘉祐二年 (1057)·苏轼 / 苏辙 / 曾巩 / 程颢 / 程颐 / 张载 / 吕惠卿同年·**史称"千年龙虎榜"** (此榜后出 9 宰相·影响 60 年)
- 明·房师·会试 18 房·每房 1 同考官·进士入门 = 主考 + 房师 + 总裁三层座师
- 清·房师 + 总裁 + "同年录"·朝廷不禁但实际终生互助

**v7 实现锚**·Slice C1 主考 4 属性派生 = 上表 4 维·Slice E1 `_mentorRef` 写硬指向·Slice F1 `GM._discipleGraph` byMentor / byCohort 双索引·Slice F2 同年门生联动 event (此 4 slice **必须一起设计**·否则门生网络空心)。

### §1.D·干预 vs 反干预·5 反舞弊 toggle

科举从初设那天就有舞弊·朝廷反舞弊·历史上 5 条 toggle·**各朝代陆续加上·一旦加就不再废**·

| toggle | 机制 | 历史首见 | 全面 |
|---|---|---|---|
| **糊名** | 卷首考生姓名糊纸封 | 唐武则天 (殿试·初) | 北宋淳化三年 992 (省试) |
| **誊录** | 朱卷红字抄一遍·主考看誊本 | 北宋大中祥符八年 1015 (真宗) | 北宋天圣四年 1026 (仁宗·确立) |
| **锁院** | 主考定后立即锁宫·与外断绝 | 北宋淳化三年 992 | 北宋庆历元年 1041 (仁宗·制度化) |
| **回避** | 主考子弟 / 同乡 / 门生不入考·"别头试" | 唐 (亲族别头) | 北宋开宝三年 970 (太祖·扩大) |
| **监临** | 御史 / 监察御史驻场监考 | 唐 (御史台监) | 北宋·明清扩 (内监 + 外监) |

**5 toggle 同时 on**·北宋仁宗朝 (1041 后) 到清光绪·近 900 年·这是历史上**反舞弊机制最稳定的窗口**。元代有过部分关闭 (色目人特权)·明末有过松弛 (魏忠贤 1624-1627 阅卷干预·见 §6)。

**反干预的反向**·5 toggle 反作用是**主考权力被削弱**·因主考不知考生身份·只能凭文章。这反过来催生 2 个 paradigm·
1. **文风固化**·没人脸辨识·主考用"文风"辨人 (北宋欧阳修排太学体·南宋朱熹尊四书·明八股)·这是**学派渗透科举的窗口**·见 §1.C
2. **行卷 / 关节**·考前递文章 (唐风)·考时递关节字 (北宋后兴)·清末"关节条子"是 toggle 时代的舞弊新形态

**v7 实现锚**·5 toggle 是 **scenario.kejuFlags** 字段 (Slice K2 编辑器)·各朝代默认值见 §2·玩家可在 Slice A1 制度激活时调整·关 toggle = 主考偏 + 弊案率 + huangwei 抗议·开 toggle = 文风固化 + 学派渗透加强 (Slice C1)。

### §1.E·身份金字塔·4-5 级身份阶梯

进士不是终点·**进士内部还有 3-4 级身份**·这是 v7 殿试 + 朝考机制的依据·

通用层级 (明清最完整·宋唐简化版)·
1. **进士及第** (一甲·状元 / 榜眼 / 探花)·**每科 3 人**·直入翰林
2. **进士出身** (二甲·若干人)·朝考可入翰林为庶吉士
3. **同进士出身** (三甲·若干人)·入吏部铨选·外放知县
4. **庶吉士** (二甲 + 三甲优秀者朝考选)·翰林院学习 3 年·"散馆"后留翰林 / 外放
5. **翰林** (一甲 + 庶吉士留馆者)·**未来宰相 / 大学士的预备役**·明清"非翰林不入内阁"

**朝代差异**·
- 唐·只一甲 (状元 + 榜眼 + 探花)·二三甲不分·**及第直接释褐**
- 宋·一甲 + 二甲 + 三甲分立·但**无朝考 / 庶吉士**·进士直接授官 (状元授通判·余授知县 / 节度推官)
- 明·**朝考 + 庶吉士确立** (1404 永乐二年)·"非翰林不入内阁"成铁律
- 清·袭明制·加"散馆考"·庶吉士 3 年后再考·留馆 / 改部 / 外放

**名臣对照** (明清宰辅 90% 翰林)·
- 张居正·1547 进士二甲 → 庶吉士 → 内阁首辅
- 戚继光·武进士 (1556 庶常未中)·终身武职·**非翰林·不能入阁**
- 曾国藩·1838 进士三甲 → 朝考一等 → 庶吉士 → 留馆 → 内阁
- 左宗棠·**举人** (1832)·三次会试不中·靠军功封侯·终身耿耿"未中进士"

**v7 实现锚**·Slice D1 殿试代主 6 身份·Slice E1 授官分配·Slice E2 庶吉士选拔 (现有 ~)·Slice F1 cohort 跟踪·都按此 4-5 级走。`scenario.dynasty` 决定开启哪几级 (唐宋简化·明清全开)。

### §1.F·历史功能·F1 士人吸纳 / F2 官僚流动 / F3 文化整合

科举的**社会功能**·1300 年都是这 3 个·v7 Slice J1 公式化·

- **F1·士人吸纳率**·把读书人引向考场而非作乱。F1 高·士人安定·F1 低·士人转向私学 / 党社 / 起义。历史数值估·北宋 ~80 (最高)·元 ~30 (最低)·明清 ~60-70·清末 ~40。
- **F2·官僚流动率**·新进士 / 总官员。F2 高·官僚循环快·F2 低·僵化。历史数值估·北宋 ~50 (3 年 1 科 + 200-400 录·循环最快)·明清 ~30-40 (录额相同但官位累积)·清末 ~20。
- **F3·文化整合度**·偏远地区进士占比 + 解额公平度 + 门生网络多样性。F3 高·王朝凝聚·F3 低·分裂。历史数值估·北宋 ~70 (川闽湘崛起)·南宋 ~50 (北方丢)·元 ~40 (色目压汉)·明 ~60 (南北分卷)·清 ~55。

**v7 实现锚**·Slice J1 公式 + endTurn 钩子·见 sprint plan C.1/C.2/C.3。本 doc 提供**历史数值锚** (上行) 用于 calibrate·跑公式后若 1080 北宋 F1 < 70·公式必有 bug。
